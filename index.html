Java Tribute

What is Java?

"Java is a programming language and computing platform first released by Sun Microsystems in 1995. It has evolved from humble beginnings to power a large share of today's digital world, by providing the reliable platform upon which many services and applications are built. New, innovative products and digital services designed for the future continue to rely on Java, as well."

Key Features

  • Reliable
  • Portable
  • Secure
  • Built in garbage collector

Timeline

1995
JDK Beta is released.Developed by James Gosling at Sun Microsystems.
1996
Java's first public release: Java 1.0.2. The release was titled Java 1.
2004
For-each loop is added to Java along with Autoboxing and Generics API.
2011
Minecraft, which was built with Java, is released to huge sucess.
2019
Switch Expressions, Textblocks and Unicode 12 support is added.
